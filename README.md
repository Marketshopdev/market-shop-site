# 🛒 Market Shop - Site Vitrine Professionnel

> **Statut du projet :** Démo en cours de développement 🚀
> **Développeur :** DJ PADESCO (Padesco Digital Studio)

Ce projet est un site vitrine multi-pages moderne, fluide et entièrement responsive conçu pour **Market Shop**, afin de présenter ses produits, services et de faciliter la prise de contact avec ses clients.

---

## 🛠️ Technologies Utilisées

*   **HTML5** : Structure sémantique et propre.
*   **CSS3** : Design moderne avec variables globales (thème Bleu Foncé, Blanc et Orange)[span_0](start_span)[span_0](end_span).
*   **JavaScript (Vanilla)** : Code modulaire et performant, découpé en fonctionnalités indépendantes pour une maintenance simplifiée.

---

## 📂 Structure du Projet

Le projet applique une architecture modulaire et ordonnée pour garantir un code propre et facilement modifiable :

```text
market-shop/
├── index.html          # Page d'accueil
├── about.html          # Page À propos
├── products.html       # Page d'exposition des produits
├── services.html       # Page de présentation des services
├── gallery.html        # Galerie photos
├── contact.html        # Formulaire et coordonnées
├── README.md           # Documentation du projet (ce fichier)
├── css/
│   ├── style.css       # Styles généraux et variables de couleurs
│   └── components.css  # Styles des composants réutilisables (boutons, etc.)
└── js/
    ├── main.js         # Orchestrateur et initialisation du site
    ├── youtube.js      # Gestion indépendante de l'intégration vidéo
    └── contact.js      # Gestion indépendante du formulaire de contact
